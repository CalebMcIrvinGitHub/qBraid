qbraid envs
============

List aliases of all installed virtual environments.

.. code-block:: bash

    qbraid envs


Examples
---------

.. code-block:: console

    $ qbraid envs
    The environments you have installed are:
    aws_braket
    qsharp
    qbraid
    qbraid_sdk
    To install a package run qbraid install (<package> <environment>)
