qbraid envs list
=================

Get list of installed qBraid environments.

.. code-block:: bash

    qbraid envs list


Examples
---------

.. code-block:: console

    $ qbraid envs list
    Installed environments:
    qsharp
    default
    qiskit
    amazon_braket
    qbraid_sdk
    Use `qbraid envs -h` to see available commands.
